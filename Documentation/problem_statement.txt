# Professional Project Proposal

## **Project Title**
**Explainable Hybrid Quantum-Classical Network Intrusion Detection System Using Variational Quantum Circuits**

---

## **Problem Statement**

Modern cybersecurity systems face unprecedented challenges in detecting sophisticated network intrusions due to the increasing complexity and volume of attack patterns. Traditional Machine Learning-based Network Intrusion Detection Systems (NIDS) suffer from three critical limitations:

1. **Black-box Decision Making**: Classical ML models lack transparency in their detection mechanisms, making it difficult for security analysts to understand and trust automated threat assessments.

2. **Computational Constraints**: High-dimensional network traffic data requires computationally expensive feature extraction and pattern recognition, limiting real-time detection capabilities.

3. **Poor Generalization**: Existing approaches show significant performance degradation on complex, modern attack datasets (e.g., 64% accuracy on UNSW-NB15 dataset in recent quantum-enhanced IDS research).

**Research Gap**: While quantum computing promises exponential speedup in certain ML tasks, existing quantum-enhanced NIDS implementations (Gouveia & Correia, 2020) utilize static quantum kernels without trainable parameters and provide no mechanism for explainability—a critical requirement in security operations where false positives can be costly and false negatives catastrophic.

---

## **Proposed Solution**

We propose a **novel hybrid quantum-classical architecture** that addresses these limitations through three key innovations:

### **1. Variational Quantum Circuits (VQC) with Trainable Parameters**
Unlike prior work using fixed quantum feature maps, our approach employs parameterized quantum circuits that learn optimal data representations through gradient-based optimization, enabling adaptive pattern recognition for diverse attack types.

### **2. Explainable AI Integration**
We integrate SHAP (SHapley Additive exPlanations) analysis to provide transparent, feature-level explanations for detection decisions, bridging the trust gap between automated quantum systems and human security analysts.

### **3. Advanced Feature Engineering**
We employ Variational Autoencoders (VAE) for probabilistic dimensionality reduction, specifically addressing the distribution mismatch issues that caused 28% accuracy degradation in prior quantum NIDS research on the NB15 dataset.

---

## **Research Objectives**

### **Primary Objectives**
1. **Achieve ≥80% accuracy on UNSW-NB15 dataset** (vs. 64% in baseline quantum approach)—demonstrating 25% relative improvement through optimized quantum circuits and feature engineering.

2. **Implement transparent decision-making framework** using SHAP/LIME to provide interpretable explanations for threat classifications, addressing the explainability gap in quantum-enhanced security systems.

3. **Validate on modern attack datasets** (CICIDS2017) to demonstrate generalization beyond legacy benchmark datasets (NSL-KDD), proving practical applicability to contemporary threat landscapes.

### **Secondary Objectives**
4. **Design scalable hybrid architecture** that optimally allocates computational tasks between quantum and classical processors based on pattern complexity.

5. **Benchmark quantum advantage** through comparative analysis identifying specific attack types where quantum processing provides measurable performance gains over classical methods.

6. **Enable multi-class classification** beyond binary normal/anomalous detection to identify specific attack categories (DoS, Probe, R2L, U2R, etc.).

#### Week 11-12: Hybrid Architecture & Multi-Class Classification
**Objectives:**
- Build ensemble system combining quantum and classical strengths
- Extend beyond binary classification to attack type identification

**Tasks:**
- Hybrid ensemble implementation:
  ```python
  class HybridEnsemble:
      def __init__(self):
          self.quantum_vqc = VQC(...)      # Complex patterns
          self.classical_rf = RandomForest() # Fast baseline
          self.classical_xgb = XGBoost()   # High accuracy
          
      def predict(self, X):
          q_pred = self.quantum_vqc.predict_proba(X)
          rf_pred = self.classical_rf.predict_proba(X)
          xgb_pred = self.classical_xgb.predict_proba(X)
          
          # Optimized weighted voting
          return 0.5*q_pred + 0.3*xgb_pred + 0.2*rf_pred
  ```
